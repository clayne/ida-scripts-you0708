# IDA
A place for IDA related stuff.

## idapython_tools
My tiny IDAPython tools.

### FindCrypt
A Python implementation of FindCrypt plugin.

### 695 to 7
IDAPython API name converter.

### FindGUID
GUID resolver.

### Hide Junk Code
An IDAPython script to hide junk code. Run hide_junk_code.py on IDA.
